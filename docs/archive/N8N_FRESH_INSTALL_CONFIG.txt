🚀 PASOS PARA APLICAR EN COOLIFY:
================================================================

1. ELIMINAR SERVICIO ANTIGUO:
   - Ve a Coolify → n8n → Delete/Remove
   - Confirma la eliminación
   - Espera que se elimine completamente

2. CREAR NUEVO SERVICIO:
   - En Coolify, click en "New Resource" o "Add Service"
   - Busca "n8n" en la lista de servicios
   - O selecciona "Docker Image" y usa: docker.n8n.io/n8nio/n8n:latest

3. CONFIGURAR BÁSICO:
   - Name: n8n
   - Domain: n8n.chamosbarber.com
   - Port: 5678

4. AGREGAR VARIABLES DE ENTORNO:
   - Ve a "Environment Variables"
   - Copia TODAS las variables de arriba
   - Pégalas una por una en Coolify
   - O si Coolify permite, pega todo el bloque de una vez

5. CONFIGURAR RED:
   - SSL/HTTPS: Habilitado
   - Auto SSL: Habilitado (Let's Encrypt)
   - Puerto expuesto: 5678

6. DEPLOY:
   - Click en "Deploy" o "Start"
   - Espera 2-3 minutos
   - Verifica los logs para asegurarte que no haya errores

7. VERIFICAR:
   - Abre: https://n8n.chamosbarber.com
   - Deberías ver la pantalla de configuración inicial de n8n
   - Si habilitaste Basic Auth, te pedirá usuario/contraseña
   - Crea tu cuenta de usuario en n8n

🔍 TROUBLESHOOTING:
================================================================

Si n8n no arranca:

1. Revisa los logs en Coolify → n8n → Logs
2. Busca errores de:
   - "database connection failed" → Verifica que PostgreSQL esté corriendo
   - "authentication failed" → Verifica credenciales
   - "port already in use" → Asegúrate que el servicio viejo esté eliminado

3. Verifica que el servicio PostgreSQL esté corriendo:
   - En Coolify, debería haber un servicio "postgres" o similar
   - Debe estar en estado "Running"

4. Si sigue sin funcionar:
   - Stop n8n
   - Start PostgreSQL primero
   - Luego Start n8n
   - Espera 2-3 minutos
