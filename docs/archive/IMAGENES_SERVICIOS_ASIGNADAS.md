## 🔧 Actualizar Imágenes en el Futuro

Para actualizar las imágenes de los servicios, ejecuta:

```bash
cd /home/user/webapp
node scripts/update-servicios-imagenes.js
```

O edita manualmente desde el Panel Admin:
1. Ve a Admin → Servicios
2. Click en "Editar" en cualquier servicio
3. Sección "Imagen del Servicio"
4. Sube nueva imagen o cambia URL
